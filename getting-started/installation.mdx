# Installation

Get Cocobase running in your project in minutes. Choose your language and follow the simple setup steps.

## Prerequisites

Before installing Cocobase, make sure you have:

- An account at [cocobase.buzz](https://cocobase.buzz)
- Your project API key (found in your dashboard)
- A package manager installed (npm, yarn, pub, or go modules)

## Install the SDK

Choose your preferred language and install the Cocobase SDK:

<Tabs>
  <Tab title="JavaScript">
    ```bash
    # Using npm
    npm install cocobase

    # Using yarn
    yarn add cocobase

    # Using pnpm
    pnpm add cocobase
    ```
  </Tab>

  <Tab title="Dart">
    ```yaml
    # Add to pubspec.yaml
    dependencies:
      cocobase: ^1.0.0
    ```
    
    ```bash
    # Then run
    flutter pub get
    
    # or for Dart projects
    dart pub get
    ```
  </Tab>

  <Tab title="Go">
    ```bash
    # Using go get
    go get github.com/lordace-coder/cocobase-go
    
    # Update dependencies
    go mod tidy
    ```
  </Tab>


</Tabs>

## Initialize Cocobase

After installation, initialize Cocobase in your project with your API key:

<Tabs>
  <Tab title="JavaScript">
    ```typescript
    import { Cocobase } from "cocobase";

    const db = new Cocobase({
      apiKey: "your-api-key-here",
    });
    ```
  </Tab>

  <Tab title="Dart">
    ```dart
    import 'package:cocobase/cocobase.dart';

    void main() async {
      // Initialize Cocobase
      final db = Cocobase(apiKey: 'your-api-key');
      
      // You're ready to use Cocobase!
    }
    ```
  </Tab>

  <Tab title="Go">
    ```go
    package main

    import (
        "github.com/lordace-coder/cocobase-go/cocobase"
    )

    func main() {
        client := cocobase.NewClient(cocobase.Config{
            APIKey: "your-api-key-here",
        })
        
        // You're ready to use Cocobase!
    }
    ```
  </Tab>


</Tabs>

## Configuration Options

The Cocobase client accepts these configuration options:

- **`apiKey`** (required) - Your project API key from the dashboard
- **`baseUrl`** (optional) - Custom API endpoint (defaults to `https://api.cocobase.buzz`)
- **`timeout`** (optional) - Request timeout in milliseconds (defaults to 30000)

> [!TIP]
> Store your API key in environment variables, not in your code. Use `.env` files or your framework's environment configuration.

## Environment Variables

Set up your API key using environment variables for better security:

<Tabs>
  <Tab title="JavaScript">
    ```bash
    # .env file
    VITE_COCOBASE_API_KEY=your_api_key_here
    ```
    
    ```typescript
    // Usage in code
    import { Cocobase } from "cocobase";

    const db = new Cocobase({
      apiKey: process.env.COCOBASE_API_KEY,
    });
    ```
  </Tab>

  <Tab title="Dart">
    ```bash
    # .env file
    COCOBASE_API_KEY=your_api_key_here
    ```
    
    ```dart
    // Load environment variables (using flutter_dotenv)
    import 'package:flutter_dotenv/flutter_dotenv.dart';
    
    await dotenv.load();
    final db = Cocobase(apiKey: dotenv.env['COCOBASE_API_KEY']!);
    ```
  </Tab>

  <Tab title="Go">
    ```bash
    # .env file or export
    export COCOBASE_API_KEY=your_api_key_here
    
    # Windows (PowerShell)
    $env:COCOBASE_API_KEY="your_api_key_here"
    ```
    
    ```go
    import "os"
    
    client := cocobase.NewClient(cocobase.Config{
        APIKey: os.Getenv("COCOBASE_API_KEY"),
    })
    ```
  </Tab>


</Tabs>

## Verify Installation

Test your installation by creating a simple document:

<Tabs>
  <Tab title="JavaScript">
    ```typescript
    async function testConnection() {
      try {
        // Create a test document
        const result = await db.createDocument("test_collection", {
          message: "Hello, Cocobase!",
          timestamp: new Date().toISOString(),
        });

        console.log("‚úÖ Connected to Cocobase!");
        console.log("Created document:", result);
      } catch (error) {
        console.error("‚ùå Connection failed:", error);
      }
    }

    testConnection();
    ```
  </Tab>

  <Tab title="Dart">
    ```dart
    void testConnection() async {
      try {
        // Create a document
        final user = await db.createDocument('users', {
          'name': 'John Doe',
          'email': 'john@example.com',
        });

        print('‚úÖ Connected to Cocobase!');
        print('Created user: ${user.id}');

        // List documents
        final users = await db.listDocuments('users');
        print('Total users: ${users.length}');
      } catch (error) {
        print('‚ùå Connection failed: $error');
      }
    }
    ```
  </Tab>

  <Tab title="Go">
    ```go
    package main

    import (
        "context"
        "fmt"
        "log"

        "github.com/lordace-coder/cocobase-go/cocobase"
    )

    func main() {
        client := cocobase.NewClient(cocobase.Config{
            APIKey: "your-api-key-here",
        })

        ctx := context.Background()

        // Try to list documents
        docs, err := client.ListDocuments(ctx, "users", nil)
        if err != nil {
            log.Printf("‚ùå Connection failed: %v\n", err)
            return
        }

        fmt.Printf("‚úÖ Connected to Cocobase! Found %d documents\n", len(docs))
    }
    ```
  </Tab>


</Tabs>

## TypeScript Support

For JavaScript/TypeScript projects, Cocobase includes full type definitions. No additional `@types` packages needed.

```typescript
// Types are automatically available
import { Cocobase, CocobaseConfig } from "cocobase";
```
